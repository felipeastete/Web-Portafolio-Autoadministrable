# Portafolio Web Profesional Autoadministrable

## Descripción General

Este proyecto corresponde al desarrollo de un sistema web dinámico denominado:

# “Portafolio Web Profesional Autoadministrable”

El sistema fue desarrollado como parte de la Evaluación N°3 del módulo de Desarrollo y Diseño Web.

La aplicación permite presentar información profesional de manera moderna, responsive y administrable mediante tecnologías frontend y backend.

---

# Objetivo del Proyecto

Diseñar e implementar un portafolio profesional dinámico utilizando tecnologías web modernas, aplicando buenas prácticas de desarrollo frontend y backend, integración con bases de datos y diseño responsive basado en wireframes previamente desarrollados.

# Instalación Local

## Requisitos

* PHP 8+
* MySQL
* Apache
* XAMPP o servidor compatible

## Pasos

1. Clonar repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

2. Importar `bd.sql` en MySQL

3. Configurar conexión en:

```plaintext
/config/conexion.php
```

4. Ejecutar proyecto en navegador
